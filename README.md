Homomorphic Voting System 🗳️

A secure online voting system using homomorphic encryption.

🔐 Features
- Ensures privacy and integrity
- Flask-based frontend
- End-to-end encrypted vote processing

🛠 Tech Stack
- Python
- Flask
- Cryptography

🚀 Run Locally
1. Clone the repo  
2. Install requirements  
3. Run `app.py`
